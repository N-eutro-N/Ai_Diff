## Описание проекта
Реализация лабораторной работы №2 по дисциплине "Программирование на Python". 
Проект включает решение 8 задач для варианта 8 по работе с встроенными типами данных Python.

## Требования
- Python 3.8 или выше
- Стандартные библиотеки Python (без дополнительных зависимостей)

### Реализованные задачи (Вариант 8)

Задача 1: Вычисление выражения "too soon to tell".replace('o','*').replace('* ','')

Лучшее решение: ChatGPT (минималистичное)


Задача 2: Вычисление выражения [115,202,192,334,257][:4]

Лучшее решение: Claude (с подробным объяснением)


Задача 3: Конвертация строки в верхний/нижний регистр

Лучшее решение: DeepSeek AI 


Задача 4: Преобразование списка чисел в одно число

Лучшее решение: ChatGPT (питоническое)


Задача 5: Проверка существования ключа в словаре

Лучшее решение: DeepSeek AI 


Задача 6: Нахождение длины множества двумя способами

Лучшее решение: GitHub Copilot (3 способа)


Задача 7: Работа с файлами + поиск самого частого слова

Лучшее решение: DeepSeek AI 


Задача 8: Работа со словарями через модуль pickle

Лучшее решение: GitHub Copilot (комплексное)
